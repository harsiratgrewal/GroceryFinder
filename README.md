# GroceryFinder
Grocery finder will help the consumer by finding the cheapest, fastest, or most efficient (cheapest and fastest) shopping options for the selected products that they need to get. 

IDE Used: VSCode
Programming Language: JavaFX
GUI Design: Scene Builder

Program is dependent on active SQL database, GoogleMaps API, and Kroger API token 
Following text files should exist in the project folder:
APIinfo.txt - Need API key for accessing GoogleMaps
dbInfo.txt - Need database info, url, username, password
krogerInfo - Need token authorization to access Kroger API's

This project currently only includes Kroger stores API for location and product data. More stores are added with different API response, but all with be integrated into
the database the same way.
